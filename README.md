# agents-conventions

Shared source of truth for GitHub-focused agent conventions, instructions, and workflows.

- Collect reusable coding conventions for GitHub-based projects.
- Publish shared instruction blocks for agents and CI-assisted workflows.
- Keep generic guidance here instead of duplicating it across repos.
- Support selective import of tagged conventions into consuming repositories.

Use these for verification and testing of conventions, and as examples for new consumers.
