# B.S.T. Band Website

## Preface

This project generates the band website of the German doom metal band B.S.T.
The band is based in Hamburg, Germany. Lyrics and music are in German, as is the Website (except
for the international band info page, which is in English.). The language of this project is English.

## Code Setup

* use kotlin multi language
* use latest gradle
* use [Kobweb](https://github.com/varabyte/kobweb)
